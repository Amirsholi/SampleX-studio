SAMPLEX — DEMO TRIAL VERSION

La carpeta que debes importar en Chrome se llama "SampleX". Este README queda fuera de ella a propósito.

INSTALACIÓN

1. Descomprime el archivo ZIP completo.
2. Abre chrome://extensions en Google Chrome.
3. Activa "Modo desarrollador".
4. Pulsa "Cargar extensión descomprimida".
5. Selecciona únicamente la carpeta "SampleX" que quedó al descomprimir.
6. Fija SampleX en la barra de herramientas y ábrelo sobre la pestaña que quieras capturar.

La Demo Trial incluye 75 exportaciones WAV. La grabación, el recorte y el análisis se procesan localmente en tu navegador.
