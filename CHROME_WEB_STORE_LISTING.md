## Detailed description

SampleX turns audio playing in your active Chrome tab into a clean, editable WAV sample.

Record only when you choose, select the useful region, trim the result, inspect BPM, musical key and frequency information, then export a WAV file without uploading the recording to a remote processing server.

Features:

- User-initiated active-tab audio recording
- Visual waveform selection and trimming
- Local BPM, key and frequency analysis
- WAV export
- 75 free exports
- Optional 500-export packs and a lifetime unlock
- Signed offline license activation

SampleX is intended for audio you own, created, licensed, or have permission to record. You are responsible for respecting copyright, website terms and applicable law.

Audio processing happens locally in the browser. See the privacy policy for complete data-handling details.
